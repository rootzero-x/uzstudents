# UzStudents Platform — Project Context

## Project Name
UzStudents

## Description
UzStudents — bu online masofaviy ta’lim platformasi.
Universitetda sirtqi / masofaviy ta’limda o‘qiydigan talabalar uchun mo‘ljallangan.

Talabalar:
- Mustaqil ishlarni yuklaydi
- Topshiriqlarni platforma orqali topshiradi

O‘qituvchilar:
- Topshiriqlarni ko‘radi
- AI yordamida avtomatik tekshiruv va baholash oladi

## Tech Stack
Frontend:
- React (Vite)
- TailwindCSS
- JavaScript (keyin TypeScript o‘ylab ko‘riladi)

Backend (rejalashtirilgan):
- Node.js / Express yoki NestJS
- PostgreSQL yoki MongoDB
- File storage (S3 yoki local)

AI Integration:
- OpenAI API
- AI assignment checking
- AI scoring + feedback

## Design Workflow
- Dizaynlar screenshot ko‘rinishida yuboriladi
- ChatGPT dizaynga qarab React + TailwindCSS code yozadi
- UI componentlar reusable bo‘ladi

## Coding Rules
- Har bir section alohida component
- UI elementlar (`Button`, `Input`, `Card`) umumiy
- Hardcoded textlar `data/` papkada
- Clean, scalable, production-ready code

## Pages (Planned)
- Home
- Courses
- Course Detail
- Auth (Login / Register)
- Student Dashboard
- Teacher Dashboard
- Assignment Upload
- AI Review Result Page
